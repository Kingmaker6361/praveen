# WhatsApp Clone (Flask + SocketIO)

Features:

* User Registration
* Login
* Real-Time Messaging
* Private Chats
* SQLite Database
* Image Upload
* Video Upload
* Document Upload
* Online Status

## Installation

pip install -r requirements.txt

## Run

python app.py

Open:

http://127.0.0.1:5000

## Project Structure

app.py
